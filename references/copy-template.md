# 推广文案模板 (copy.md)

## 格式说明

每张贴图的推广文案格式为：

```markdown
## {贴图名称}
### 内容
{推广文案，20-50字左右}

### 标签
#标签1 #标签2 #标签3 #标签4 #标签5
```

## 字段要求

| 字段 | 级别 | 要求 | 说明 |
|-----|------|------|------|
| 贴图名称 | H2 | 不超过20个字 | 如"摸鱼中"、"太强了" |
| 内容 | H3 | 20-50字 | 推广文案，适合社交媒体发布 |
| 标签 | H3 | 至少5个 | 空格分隔，以 # 开头 |

## 文案风格指南

### ✅ 推荐写法
- 简短有力，突出贴图特色（15-30字）
- 社交友好，适合朋友圈传播
- 引发共鸣，触达用户痛点
- 包含互动引导

### ❌ 避免写法
- 过于冗长（超过50字）
- 纯产品介绍，无情感连接
- 使用生硬的专业术语
- 与贴图内容无关

## 标签分类（至少选5类）

| 类别 | 说明 | 示例 |
|-----|------|------|
| 平台标签 | 标注平台属性 | #微信表情 #微信贴图 #WeChatStickers |
| 情感标签 | 表达情感类型 | #可爱 #搞笑 #治愈 #社恐自救 |
| 主题标签 | 贴合内容主题 | #摸鱼 #打工人 #职场 #AI #编程 |
| 风格标签 | 描述视觉风格 | #赛博朋克 #卡通 #简约 #手绘 |
| 用途标签 | 使用场景 | #斗图 #表情包 #头像 #每日壁纸 |

## 输出规范

- **文件名**: `copy.md`
- **位置**: 项目根目录，与 `sticker-manifest.md` 同级
- **格式**: Markdown
- **编码**: UTF-8
- **数量**: 每个贴图一个章节，至少3张贴图

## 注意事项

1. **标题字数**：严格控制在20个字以内
2. **内容长度**：建议20-50字，不超过100字
3. **标签数量**：至少5个，推荐6-10个
4. **标签格式**：空格分隔，无需逗号
5. **顺序排列**：建议按平台→情感→主题→风格→用途
